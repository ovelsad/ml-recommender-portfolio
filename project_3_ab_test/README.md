# Project 3 — A/B тестирование рекомендательных моделей

## Задача
Сравнение двух рекомендательных моделей (контрольной и тестовой) через A/B эксперимент с хешированием пользователей.

- FastAPI сервис возвращает exp_group и рекомендации
