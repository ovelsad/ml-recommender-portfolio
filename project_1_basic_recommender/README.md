# 🔄 Project 1 — Базовая рекомендательная система

## Задача
Создание сервиса, который возвращает персонализированные рекомендации постов на основе профиля пользователя и содержимого постов.

## Методы
- TF-IDF для текстов постов
- SVD для снижения размерности
- CatBoostClassifier
- FastAPI-сервис для выдачи рекомендаций

## Метрика
- Hitrate@5
- Цель: обеспечить высокое покрытие рекомендаций

## Файлы
- `download_db_TF_IDF_1_version.py` — загрузка данных, обработка текста, генерация признаков
- `ml_app_1_version.py` — FastAPI сервис, использующий модель `CatBoost_model.cbm`
